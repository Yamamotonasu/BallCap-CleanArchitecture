# CloudFirestore Test Repository with CleanArchitecture

## Requirement
- iOS 12.0
- Swift 5.1
- Xcode 11.1
- Cocoapods 1.8.4
- Carthage 0.33.0

## tips
The carthage library is under git management, so no prior operation is required.
For cocoapods, need to type the following command on the command line
  
```
$ pod install
```
